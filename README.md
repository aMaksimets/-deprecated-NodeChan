<h1 align="center">NodeChan</h1>

Installation
------------
Get [node.js + npm](https://wsvincent.com/install-node-js-npm-windows/).

1. Clone git repo

2. Inside the NodeChan folder
```bash
node app.js
```
You can access the site at your [localhost:3000](http://localhost:3000/).
